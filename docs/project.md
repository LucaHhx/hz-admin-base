# 项目概览

## 基本信息

| 字段 | 内容 |
|------|------|
| 项目名称 | hz-admin-base |
| 描述 | 基于 GVA (gin-vue-admin) 框架的后台管理基础项目，提供开箱即用的用户管理、权限控制、代码生成等通用后台管理能力，可作为业务系统的基础底座快速扩展 |
| 创建日期 | 2026-03-06 |
| 当前版本 | 2.7.9 |

## 产品核心价值

- 提供完整的后台管理基础设施，避免每个新项目从零搭建用户、权限、菜单等通用模块
- 内置代码生成器，通过可视化配置快速生成 CRUD 模块，大幅提升业务功能开发效率
- 支持多运行模式（后台管理 / API 服务 / 全部），灵活适配不同部署场景
- 基于 RBAC 的细粒度权限控制（角色、菜单、按钮、列级别），满足企业级权限管理需求

## 目标用户

- 内部开发团队：以此项目为基础底座，快速搭建各类业务后台管理系统
- 系统管理员：通过后台界面管理用户、角色、权限、系统配置等
- 业务运营人员：使用基于此底座构建的业务管理功能

## 关键实体

| 实体 | 说明 |
|------|------|
| 用户 | 系统使用者，支持注册、登录、个人信息管理 |
| 角色 (Authority) | 权限角色，支持层级关系和细粒度权限分配 |
| 菜单 | 前端导航菜单，与角色关联实现动态菜单 |
| API 接口 | 后端接口资源，通过 Casbin 实现基于角色的访问控制 |
| 字典 | 系统数据字典，用于管理下拉选项等枚举数据 |
| 操作记录 | 用户操作日志，用于审计追踪 |

## 信息架构

- **系统管理**: 用户管理、角色管理、菜单管理、API 管理、数据字典
- **权限控制**: Casbin RBAC、按钮级权限、列级权限、数据过滤
- **开发工具**: 代码生成器（自动化 CRUD）、代码生成历史、插件管理
- **通用功能**: 文件上传下载、导出模板、操作记录、参数配置、国际化翻译
- **系统运维**: JWT 鉴权、IP 限流、系统配置、健康检查
- **插件系统**: 公告管理、邮件服务

## 成功指标

- 新业务模块从零到可用的搭建时间不超过 1 天（借助代码生成器）
- 基础权限体系开箱即用，无需业务项目重复实现
- 系统稳定运行，支持多数据库（MySQL / PostgreSQL / SQLite / MSSQL / Oracle）
- 模板项目开箱即用，默认使用 SQLite，Redis 为可选组件（非强制依赖）

## 目录结构

```
/
├── docs/           # 项目文档
├── server/         # Go 后端 (Gin + GORM)
│   ├── api/v1/     # API 接口层（system / example / business / api）
│   ├── model/      # 数据模型
│   ├── service/    # 业务逻辑层
│   ├── router/     # 路由定义
│   ├── middleware/  # 中间件（JWT / CORS / Casbin / 限流 / 操作记录）
│   ├── initialize/ # 初始化（数据库 / 路由 / 定时任务 / 验证器）
│   ├── plugin/     # 插件（公告 / 邮件）
│   ├── source/     # 初始化种子数据
│   └── utils/      # 工具函数
└── web/            # Vue 3 前端 (Element Plus + Vite)
```
